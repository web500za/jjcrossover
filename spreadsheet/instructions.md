# Google Sheets Submission Instructions

## Quick Setup Guide

### Step 1: Copy the Original Spreadsheet
1. Open: https://docs.google.com/spreadsheets/d/1YaimnCR6OUGKbQ9BTegmwnS4kMA7izmayjjXHs19cqo/edit?usp=sharing
2. Click: File → Make a copy
3. Rename to: "Vendor Analysis Assessment – Jared January"

### Step 2: Add Vendor Analysis Data
1. In the first sheet, add these columns after "Last 12 months Cost (USD)":
   - Column C: **Department**
   - Column D: **Vendor Description**
   - Column E: **Strategic Recommendation**
2. Copy data from `../data/vendor_analysis_complete.csv`

### Step 3: Create Top 3 Opportunities Tab
1. Create new sheet: "Top 3 Opportunities"
2. Add columns:
   - A: Opportunity Title
   - B: Description
   - C: Annual Savings (USD)
3. Content:

| Opportunity Title | Description | Annual Savings |
|-------------------|-------------|----------------|
| Salesforce License Optimization | Conduct license audit, eliminate unused seats, renegotiate contract, consolidate HubSpot | $450,000 - $650,000 |
| Office Space Consolidation | Reduce from 8+ workspace providers to 2-3 strategic partners with volume discounts | $280,000 - $380,000 |
| Professional Services Rationalization | Consolidate 10+ accounting/advisory firms to single global partner | $140,000 - $200,000 |

### Step 4: Create Methodology Tab
1. Create new sheet: "Methodology"
2. Copy content from `../analysis/methodology.md`

### Step 5: Create Recommendations Tab (Executive Memo)
1. Create new sheet: "Recommendations"
2. Copy content from `../analysis/executive_memo.md`

### Step 6: Set Permissions
1. Click: Share button
2. Change to: "Anyone with the link"
3. Set access: "Viewer"
4. Copy the shareable link
